# Pawn Simulator

## Installation

- Git clone the repo

  ```bash
  $ git clone https://github.com/vulcansmithy/the-pawn-simulator.git
  ```

- Do a bundle install

  ```bash
  $ bundle install
  ```

  

## To run

- Open up Terminal and run the following command

  ```bash
  $ ruby pawn_simulator.rb
  ```
